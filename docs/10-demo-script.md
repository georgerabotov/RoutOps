# 10 — Demo script (2 minutes)

## Pre-demo setup checklist

- [ ] **4 calendar events** today with **full London addresses** (not just venue names).
- [ ] **One intentionally tight gap** between two meetings (so the anomaly bites).
- [ ] **Home postcode** set as the day's origin (`HOME_POSTCODE`).
- [ ] **Pre-seed or live-match a TfL disruption** on one line that touches a leg.
- [ ] **Test OAuth** end-to-end on the demo account beforehand.
- [ ] **Send one test WhatsApp** message via Wassist to confirm delivery.
- [ ] Loom backup recorded in case live feeds misbehave.

## 2-minute timed script

| Time | Beat |
| --- | --- |
| **0:00** | **Pitch.** "RouteOps tells you when today is abnormal for the season — and why — before you're late." |
| **0:20** | **Calendar.** Show the 4 real meetings being read live via Google Calendar. |
| **0:40** | **Dashboard legs.** Timeline of legs with mode, leave-by, and normal-vs-today status badges. |
| **1:00** | **Verified attribution.** Click the severe leg → "Verified (3 sources)" → TfL + 2 news outlets agree. |
| **1:20** | **WhatsApp leave-by.** Show the phone: "Leave by 14:12 for your 15:00 — Central line delays add 18 min." |
| **1:40** | **Add a meeting → live recompute.** Drop in a new event; the loop re-routes and updates leave-by. |
| **1:55** | **Escalation or PayPal.** Either the "day is impossible → human escalation" flag, or one PayPal sandbox payment. |

## Hero sentence

> **"Normal for this leg is 26 minutes. Today is 48. We know why — and we told you before you were late."**
